Projects

Developing a Website for CSE Lab

I had the privilege of developing a fully functional website for the IIT Lab project, which serves as a platform for students and researchers. My goal was to create an intuitive and visually appealing interface that enhances user experience. I utilized technologies like HTML, CSS, JavaScript, and Bootstrap for the front-end, while implementing Node.js and Express.js for the back-end functionalities. The website features project listings,profiles of professors, a resource repository, and a contact desk. Through my role as the sole developer, I handled the end-to-end development process, collaborated with the IIT Lab team, and ensured a responsive design for optimal accessibility across devices.
